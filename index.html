Create a New App

These are instructions for designers who want to learn web development and React. Below are simple steps to creating your first React app.

Your app only needs one build dependency. We test Create React App to make sure that all of its underlying pieces work together seamlessly – no complicated version mismatches.

Get started

Whether you’re using React or another library, Create React App lets you focus on code, not build tools.

To create a project called my-app, run this command:

npx create-react-app my-app

Update React

Updating your build tooling is typically a daunting and time-consuming task. When new versions of Create React App are released, you can upgrade using a single command:

npm install react-scripts@latest

Select packaging manager

When you create a new app, the CLI will use npm or Yarn to install dependencies, depending on which tool you use to run create-react-app. For example:

# Run this to use npm npx create-react-app my-app # Or run this to use yarn yarn create react-app my-app

To install React and MUI, you will need to use Node.js and its package manager, npm. Here are the steps you can follow:

  1. Install Node.js: Visit the Node.js website (https://nodejs.org) and download and install the version that corresponds to your operating system. You will have to restart your computer.
  2. Install npm and npx. npm install -g npm
  3. Create a new React app: Open a terminal and run the following command: npx create-react-app my-app
  4. Navigate to the directory of your React app
  5. Install React and start the server.

    6. This will automatically open http://localhost:3000 in your default browser, and you will see your React app.

    Start building your app: Open the project in your preferred code editor and start modifying the source files. The entry point of your app is src/index.js, and you can create new components in the src directory.

    Navigate to the directory of your React app by running:

    1. cd my-app
      1. Navigate back a directory: cd ..
      2. Navigate forward: cd {next directory}
